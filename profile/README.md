# Peso for PHP

Peso is a PHP framework to query currency exchange rates from various sources like specialized currency exchange data services and central banks.

Documentation page: [phpeso.org](https://phpeso.org) \
Demo page: [demo.phpeso.org](https://demo.phpeso.org)

Rate providers:
[European Central Bank](https://github.com/phpeso/ecb-service),
[Czech National Bank](https://github.com/phpeso/cnb-service),
[Hungarian National Bank](https://github.com/phpeso/mnb-service),
[National Bank of Romania](https://github.com/phpeso/bnr-service),
[Fixer](https://github.com/phpeso/fixer-service),
[Open Exchange Rates](https://github.com/phpeso/openexchangerates-service),
[FreecurrencyAPI](https://github.com/phpeso/freecurrencyapi-service),
[CurrencyAPI](https://github.com/phpeso/currencyapi-service)

Integration packages:
[Money for PHP](https://github.com/phpeso/moneyphp-exchange),
[Brick\Money](https://github.com/phpeso/brick-rateprovider)

[Write your own Service or Integration!](https://phpeso.org/v1.x/extend.html)
